# [chess3D](https://yafred.github.io/chess3D/)

Play on Lichess using a 3D model and the Lichess public API.

## Run it on your machine

1. Clone https://github.com/yafred/chess3D 
1. `cd chess3D`
1. `pnpm install`
1. `pnpm run build`
1. `pnpm run serve`

## Vite development server 
1. `cd chess3D`
1. `pnpm run dev`

## Lichess api-demo

This app is based on [Lichess OAuth demo](https://github.com/lichess-org/api-demo)
